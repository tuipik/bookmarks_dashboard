# Start Dashboard

Простий сервіс для стартової сторінки (дашборд із колонками й картками).

Запуск локально:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

Відкрийте http://localhost:5000

Файли:
- `app.py` — Flask backend і API
- `templates/index.html` — фронтенд
- `static/*` — CSS і JS
